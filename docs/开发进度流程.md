## 3. 模块一：综合气象模型服务

### 功能说明

以 **ghr_9km** 为核心，提供简易单点、专业多变量、批量与区域三级查询，覆盖 11 天逐小时预报及历史年景对比，满足验收演示与业务查询入口要求。

### 技术实现

| 任务项 | 状态 | 产出物 |
|--------|------|--------|
| ghr_9km 模型展示与参数配置 | ✅ | constants.ts、各查询页 |
| 简易查询（地图选点+4 图表+导出） | ✅ | SimpleQueryPage.tsx |
| 专业查询（变量选择器+多曲线+表格） | ✅ | ProfessionalQueryPage.tsx |
| 批量/区域/历史年景查询 | ✅ | BatchQueryPage.tsx |
| 预报数据生成与范围校验 | ✅ | 数据层模块 |

### 对应页面/路由

| 路由 | 页面说明 |
|------|----------|
| `/simple-query` | 简易气象查询 |
| `/professional-query` | 专业气象查询 |
| `/batch-query` | 批量与区域查询 |

### 数据来源

- 简易查询预报结果
- 专业查询多变量结果
- 6 个预置批量查询坐标
- 2016–2025 历史年景数据

### 验收要点

- [ ] 三大查询路由可访问，模型名 ghr_9km 统一
- [ ] 经纬度 72°–137°E、17°–55°N 校验生效
- [ ] 专业查询最多 4 变量、13 气压层可选
- [ ] 查询图表/表格渲染正常，JSON 可导出
