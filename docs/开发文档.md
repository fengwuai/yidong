## 1. 项目概述

AI气象预测数据中心与可视化平台是一套可弹性扩展、面向业务场景优化的气象数据中心架构体系，实现 PB 级气象数据的统一接入、长期归档、快速检索、批量导出与标准化服务，并新增极端天气预警全流程功能。平台通过标准化 API、可视化交互界面及多渠道推送机制，为物流交通、新能源、电力交易、灾害预警等业务系统提供稳定、高效、安全的数据支撑与预警服务。

### 1.1 核心功能模块

| 模块 | 路由 | 说明 |
|------|------|------|
| 数据中心概览 | `/dashboard` | PB 级存储监控、冷热分层、多源接入状态、系统运行指标 |
| 简易气象查询 | `/simple-query` | 单点快速查询，11 天逐小时预报可视化 |
| 专业气象查询 | `/professional-query` | 多变量精细化查询，支持高空气压层数据 |
| 极端天气预警 | `/extreme-warning` | 实时预警监控、地图高亮、预警溯源、弹窗提醒 |
| 预警规则配置 | `/warning-rules` | 可视化规则引擎，阈值配置、逻辑组合、模拟触发测试 |
| 预警推送管理 | `/warning-push` | 多渠道推送配置、状态追踪、失败重试 |
| 数据推送与导出 | `/data-push` | 模板化任务管理，批量坐标/区域定时推送 |
| 历史任务导出 | `/history` | 推送任务执行状态，下载/重新导出 |
| API 文档 | `/api-docs` | OpenAPI 3.0 规范，气象+预警接口说明 |
| 账户与安全 | `/settings` | 权限角色、Token 管理、操作审计日志（等保三级） |

### 1.2 技术参数

| 参数项 | 说明 |
|--------|------|
| 气象模型 | ghr_9km（全球 9km 分辨率） |
| 覆盖范围 | 经度 72°–137°E，纬度 17°–55°N |
| 时间分辨率 | 逐小时预报 |

| 起报频次 | 每天 4 次（UTC 00/06/12/18 时） |

| 权限说明 | 简易查询免费；专业查询、API 调用、数据推送按点数计费 |

## 4. 功能模块详细说明

### 4.1 简易气象查询

**功能定位：** 面向快速查询、可视化预览需求的轻量化功能，操作零门槛。

**操作流程：**
1. 选择日期（支持历史及未来 10 天预报日期）
2. 选择位置：
   - 下拉选择预设城市（16 个主要城市）
   - 自定义输入经纬度（经度 72°–137°E，纬度 17°–55°N）
   - 点击地图直接选点
3. 点击「查询（免费）」按钮，系统生成逐小时气象预报图表
4. 点击「下载」按钮，导出当前查询数据（JSON 格式）

**可视化内容（11 天逐小时）：**
- 温度（℃）
- 太阳辐射（MJ/m²）
- 10m 风速 / 100m 风速（m/s）
- 降水（mm）

**相关文件：** `src/pages/SimpleQueryPage.tsx`、`src/components/Charts/WeatherCharts.tsx`、`src/components/Map/WeatherMap.tsx`

---

### 4.2 专业气象查询

**功能定位：** 面向专业气象分析、多要素深度研究的进阶功能。

**操作流程：**
1. 输入目标经纬度（范围校验）
2. 选择目标日期与预报时次（UTC 00z/06z/12z/18z）
3. 选择气象模型（默认 ghr_9km）
4. 选择气象变量（最多 4 个，支持地表参数与高空气压层变量）
5. 执行查询，查看多变量预报曲线与原始数据表格
6. 支持数据导出

**变量分类：**
- 地表参数：t2m、tcc、tp6h、ssr6h、ssr1h、sp、msl
- 风场：u10、v10、u100、v100、ws10、ws100、wd10、wd100
- 高空气压层：u/v/w/ws/wd/t/q/z + 气压层（1000–50 hPa，共 13 层）

**相关文件：** `src/pages/ProfessionalQueryPage.tsx`、`src/components/Common/VariableSelector.tsx`

---

### 4.3 数据推送与导出

**功能定位：** 面向批量数据获取、自动化定时推送的企业级功能。

#### 4.3.1 数据模板管理

模板列表展示：模板名称、气象模型、气象要素、地理位置、预估消耗点数、预估文件大小、推送状态。

模板操作：
- **编辑**：已推送过的模板不可编辑（按钮置灰）
- **下载**：手动下载对应气象数据
- **删除**：删除模板
- **开关推送**：开启/关闭定时推送任务
- **推送配置**：修改执行时次、时间间隔、数据格式、数据处理方式

#### 4.3.2 新建数据模板（三步骤）

**步骤一 — 基础信息与地理范围（三选一）：**

| 模式 | 说明 |
|------|------|
| 矩形框选 | 输入经纬度范围或在地图上拖拽框选，红色区域高亮显示 |
| 坐标列表 | 逐行输入经纬度坐标，地图自动标记位置 |
| 区域选择 | 搜索省份/城市多选，支持边界外扩与海岸线外扩（沿海区域） |

**步骤二 — 气象要素选择：**
- 支持全选/分类选择（地表参数、风场、温度、比湿、位势等）
- 支持自定义多要素、多气压层变量
- 自动统计点数消耗预计算

**步骤三 — 推送配置：**
- 执行时次：00z / 06z / 12z / 18z（可多选）
- 时间间隔：1h / 15min
- 数据格式：NetCDF / CSV
- 数据处理：原始数据 / 空间均值

**相关文件：** `src/pages/DataPushPage.tsx`

---

### 4.4 历史任务导出

**任务状态：**

| 状态 | 说明 |
|------|------|
| 排队中 | 任务已创建，等待执行时间 |
| 运行中 | 任务正在执行数据推送 |
| 已完成 | 任务运行完毕，数据已生成，可下载 |
| 失败 | 推送失败，显示失败原因，可重新导出 |

**功能：**
- 顶部统计卡片（总任务数、已完成、运行中、失败）
- 按状态筛选、关键词搜索
- 已完成任务支持下载与重新导出

**相关文件：** `src/pages/HistoryPage.tsx`

### 4.5 API 文档

提供两个开放接口的完整说明：

**接口一：单点气象查询**
```
POST https://api.weather-data.com/data_server/point
```
参数：lon、lat、vars、time、model

**接口二：历史任务查询**
```
GET https://api.weather-data.com/data_server/export_history_task/list
```
参数：is_push_gen、page

页面包含：认证说明、请求参数表、Python 示例代码（可复制）、返回示例、技术参数说明、气象变量全表。

**相关文件：** `src/pages/APIDocsPage.tsx`

## 5. 数据层设计

### 5.1 类型定义（`src/types/weather.ts`）

核心类型：`WeatherVariable`、`DataTemplate`、`HistoryTask`、`HourlyForecast`、`ProfessionalQueryResult`、`PushSchedule`、`TaskStatus` 等。

### 5.2 常量配置（`src/data/constants.ts`）

- 13 个气压层（1000–50 hPa）
- 15 个标准气象变量及分类
- 16 个预设城市
- 6 个省份及下属城市（含沿海标识）
- 图表颜色配置

### 5.3 模拟数据（`src/data/mockData.ts`）

当前版本使用前端 Mock 数据模拟 API 响应，包含：
- `generateMockWeatherResult()` — 简易查询数据生成
- `generateMockProfessionalResult()` — 专业查询数据生成
- `MOCK_TEMPLATES` — 4 条预置模板数据
- `MOCK_HISTORY_TASKS` — 8 条预置历史任务数据

> 生产环境对接时，将 Mock 函数替换为真实 API 调用即可。

### 5.4 全局状态（`src/store/weatherStore.ts`）

使用 Zustand 管理：
- 查询结果与加载状态
- 模板列表（增删改）
- 历史任务列表（更新）
- 用户 Token

## 6. 组件设计

### 6.1 布局组件

- **Layout**：左侧固定 Sidebar（240px）+ 右侧内容区
- **Sidebar**：深色导航，高亮当前路由
- **Header**：页面标题、副标题、操作按钮、剩余点数展示

### 6.2 图表组件（WeatherCharts）

| 组件 | 图表类型 | 数据字段 |
|------|----------|----------|
| TemperatureChart | 折线图 | temperature |
| RadiationChart | 面积图 | radiation |
| WindSpeedChart | 双折线图 | ws10、ws100 |
| PrecipitationChart | 柱状图 | precipitation |
| MultiVariableChart | 多折线图 | 自定义变量 |

### 6.3 地图组件（WeatherMap）

- **SimpleWeatherMap**：单点标记 + 点击选点
- **TemplateMap**：矩形框选 / 坐标标记 / 区域高亮

### 6.4 公共组件

- **VariableSelector**：地表变量 + 高空气压层变量选择，支持数量上限
- **StatusBadge**：TaskStatusBadge（排队/运行/完成/失败）、PushStatusBadge（推送中/未推送）

## 9. 支持的气象变量

| 序号 | 参数 | 单位 | 说明 |
|------|------|------|------|
| 1 | u10 | m/s | 10m U 风速（东西分量） |
| 2 | v10 | m/s | 10m V 风速（南北分量） |
| 3 | u100 | m/s | 100m U 风速 |
| 4 | v100 | m/s | 100m V 风速 |
| 5 | t2m | K | 2m 气温 |
| 6 | tcc | % | 总云量 |
| 7 | tp6h | mm | 6 小时累计降水 |
| 8 | ssr6h | J/m² | 6 小时地表太阳短波辐射 |
| 9 | ssr1h | J/m² | 1 小时地表太阳短波辐射 |
| 10 | sp | Pa | 地面气压 |
| 11 | msl | Pa | 海平面气压 |
| 12 | q[xxx]hpa | kg/kg | 比湿（如 q850hpa） |
| 13 | u[xxx]hpa | m/s | U 风速气压层（如 u500hpa） |
| 14 | v[xxx]hpa | m/s | V 风速气压层 |
| 15 | t[xxx]hpa | K | 气温气压层 |

| 18 | ws10 | m/s | 10m 风速（计算值） |
| 19 | ws100 | m/s | 100m 风速（计算值） |
| 20 | wd10 | ° | 10m 风向 |
| 21 | wd100 | ° | 100m 风向 |
| 22 | ws[xxx]hpa | m/s | 气压层风速 |
| 23 | wd[xxx]hpa | ° | 气压层风向 |

气压层支持：1000 / 925 / 850 / 700 / 600 / 500 / 400 / 300 / 250 / 200 / 150 / 100 / 50 hPa
